# 使用指南

## 知识库组织方式

本知识库按照**五层学习架构**组织内容，遵循从基础到进阶的认知规律：

```
第一层：语言基础 → 第二层：语言深入 → 第三层：框架应用 → 第四层：分布式体系 → 第五层：综合进阶
```

每一层的知识都建立在前一层的基础之上，建议按顺序学习。

## 知识条目结构

每个知识条目都遵循统一的内容结构：

1. **概念说明** — 用通俗易懂的语言解释知识点
2. **核心原理** — 深入分析底层原理，配合流程图说明
3. **代码示例** — 关键代码片段 + 完整可运行代码链接
4. **常见面试题** — 高频面试题、答题思路、深入追问

## 难度标识

每个知识条目都标注了难度级别和面试频率：

| 难度 | 说明 | 适合人群 |
|------|------|----------|
| 初级 (beginner) | 基础概念和用法 | 校招 / 初级岗位 |
| 中级 (intermediate) | 原理和源码分析 | 中级岗位 / 大厂初面 |
| 高级 (advanced) | 底层实现和架构设计 | 高级岗位 / 大厂深面 |

| 面试频率 | 说明 |
|----------|------|
| 🔥🔥🔥 高频 | 几乎每次面试都会问，必须掌握 |
| 🔥🔥 中频 | 经常出现，应该掌握 |
| 🔥 低频 | 偶尔出现，了解即可 |

## 代码示例使用

文档中的代码示例分为两种：

1. **内嵌代码片段** — 直接在文档中展示关键代码
2. **完整可运行代码** — 链接到 `code-examples/` 中的 Maven 子模块

运行完整代码示例：

```bash
cd code-examples

# 编译特定子模块
mvn compile -pl java-basics

# 运行测试
mvn test -pl java-basics

# 运行 main 方法
mvn exec:java -pl java-basics -Dexec.mainClass="com.example.basics.YourDemo"
```

## 搜索功能

点击页面顶部的搜索框（或按 `Ctrl+K` / `Cmd+K`），输入关键词即可全文搜索。

搜索范围覆盖：
- 知识条目标题
- 正文内容
- 代码注释
